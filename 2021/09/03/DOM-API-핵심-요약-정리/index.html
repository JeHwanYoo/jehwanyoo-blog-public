DOM API 핵심 요약 정리

Author: JeHwanYoo Date: 9월 3, 2021  18:58:15 Category: JavaScript
  • node, document, element, nodeList, attribute, namedNodeMap

Node

Node는 여러 가지 DOM 타입들이 상속하는 인터페이스이며 그 다양한 타입들을 비슷하게 처리할 수 있게 한다.

상속 관계

  • EventTarget ← Node ← Element ← HTMLElement

노드(Node) 생성

document.createElement() - Element

document.createTextNode() - Text

document.createAttribute() - Attr

노드(Node) 탐색

document.querySelector() - Element || null

document.querySelectorAll() - NodeList

document.getElementById() - Element || null

document.getElementsByClassName() - HTMLCollection

document.getElementsByTagName() - HTMLCollection

속성(Attribute) 접근

element.attributes - NamedNodeMap

NamedNodeMap은 index 또는 dictionary로 접근 가능
getNamedItem(name), setNamedItem(name, val), removeNamedItem(name), item(index)

element.getAttribute() - Attr

element.setAttribute()

element.getAttributeNames() - String[]

자식(child), 형제(sibling), 부모(parent) 접근

node.childNodes - NodeList

node.firstChild, node.lastChild - Node || null

node.nextSibling, node.previousSibling - Node || null

node.parentNode - Node

node.parentElement - Element

node.textContent - HTMLString

vs innerText: https://developer.mozilla.org/en-US/docs/Web/API/Node/textContent#differences_from_innertext

노드(Node) 조작

node.appendChild()

element.append(...nodesOrDOMStrings) 도 참고. 여러개의 노드를 한번에 추가 가능. (element.prepend 역시 동일)

node.clone()

deep이 true면 자식 노드까지 deep copy

node.contains()

node.insertBefore()

node.removeChild()

node.replaceChild()

엘리먼트(Element) 조작

내부 탐색

element.getElementById(), element.getElementsByClassName(), element.getElementsByTagName(), element.querySelector(), element.querySelectorAll()

element.append(…nodesOrDomStrings), element.prepend(…nodesOrDomStrings)

element.remove()

element.replaceWith()

element.after(…nodes), element.before(…nodes)

element.insertAdjacentElement(), element.insertAdjacentHTML(), element.insertAdjacentText()

element.animate(keyframe, options)
