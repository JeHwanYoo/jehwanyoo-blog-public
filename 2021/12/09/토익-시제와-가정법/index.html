[토익] 시제와 가정법

Author: JeHwanYoo Date: 12월 9, 2021  15:08:51 Category: TOEIC

참고 서적

20일만에 끝내는 해커스 토익 750+ RC

단순 시제 (현재, 과거, 미래)

  • 현재 시제는 일반적인 사실이나 일상적으로 반복되는 동작을 나타낸다.
  • 과거 시제는 과거에 이미 끝났거나 이미 끝난 동작이나 상태를 나타낸다.
  • 미래 시제는 미래 상황에 대한 예상이나 말하는 사람의 의지를 나타낸다.
  • 시간이나 조건을 나타내는 종속절에는 미래 시제 대신 현재 시제를 쓴다.

    The travel agency will send an itinerary once the accommodation is confirmed.
    If it rains tomorrow, the company anniversary will be held indoors.

진행 시제 (현재 진행, 과거 진행, 미래 진행)

  • 현재 진행 시제는 현재 시점에 진행되고 있는 일을 나타낸다.
  • 과거 진행 시제는 과거 특정 시점에 진행되고 있던 일을 나타낸다.
  • 미래 진행 시제는 미래 특정 시점에 진행되고 있을 일을 나타낸다.

완료 시제 (현재 완료, 과거 완료, 미래 완료)

  • 현재 완료 시제는 과거에 시작된 일이 현재 시점까지 계속되거나, 과거에 일어난 일이 현재까지 영향을 미칠 때, 과거의 경험에 대해 표현하거나 과거에 시작된 일이 방금 완료됨을 나타낸다.
  • 과거 완료 시제 (had + p.p.)는 과거의 특정 시점 이전에 일어난 일을 나타낸다.
  • 미래 완료 시제 (will have + p.p.)는 미래의 특정 시점 이전에 발생한 동작이 미래의 그 시점에 완료될 것임을 나타낸다.

시제 일치

  • 시제를 일치시키는데 결정적인 힌트를 제공하는 단어들
  • 과거: yesterday, recently, (시간 표현 + ago), (last + 시간 표현), (in + 과거)
  • 현재: usually, often, each month, frequently
  • 미래﹒미래 완료: tomorrow, next + 시간 표현, soon, by/util + 미래, by the time + 주어 + 현재 동사, shortly

가정법

  • 가정법 과거: 현재 사실의 반대 (~라면, ~할 텐데)
    • If + 주어 + 과거 동사, 주어 + would(could, might, should) + 동사원형
    • 조건만 되었다면 지금 당장 하고 있음을 의미한다.
  • 가정법 과거 완료: 과거 사실의 반대 (~했었더라면, ~했을 텐데)
    • If + 주어 + had p.p. ~, 주어 would have p.p.
    • 조건만 되었다면 과거에 그걸 했음을 의미한다.
  • 가정법 미래: 미래에 일어날 법한 일이나 일어날 가능성이 희박한 미래를 가정한다. (혹시 ~한다면, ~할 텐데)
    • If + 주어 + should + 동사원형, 주어 + will + 동사원형
    • 조건만 된다면 할 수 있을지도 모른다고 가정한다.

if가 생략된 가정법

  • If + 주어 + were , 주어 + would + 동사원형
    • Were + 주어 ~, 주어 + would + 동사원형
  • If + 주어 + had p.p., 주어 would have p.p.
    • Had + 주어 + p.p., 주어 would have p.p.
  • If + 주어 + should + 동사원형, 주어 + will + 동사원형
    • Should + 주어 + 동사원형, 주어 + will + 동사원형
  • 즉, 가정법에서 if가 생략되면 조동사가 앞으로 나온다.
